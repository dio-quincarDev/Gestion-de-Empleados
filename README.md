# Proyecto de Gestión de Empleados

Este es un proyecto personal de gestión de empleados desarrollado con Java 17, Spring Boot y MySQL. El objetivo es crear una aplicación completa para administrar empleados, registrar asistencia, calcular consumos y generar informes.

## Características

- Gestión de empleados (CRUD)
- Registro de asistencia
- Cálculo de consumos
- Generación de informes

## Arquitectura

El proyecto sigue una arquitectura hexagonal con las siguientes capas:

- **Adapters**: Controladores, repositorios, DTOs
- **Application**: Servicios de aplicación
- **Domain**: Modelos de dominio, servicios de dominio, excepciones
- **Ports**: Interfaces de puertos de entrada y salida
